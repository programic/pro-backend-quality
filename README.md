This package is a backend quality control

## Installation
This package requires PHP 8.1 and Laravel 10.0 or higher.

### require package
```
composer require programic/pro-backend-quality --dev
```

### install configs
```php
php artisan pro:quality:install
```

## Testing
```bash
composer test
```
